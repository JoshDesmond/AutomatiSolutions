# Agents.md

## Tooling
The application is built with Node/npm, Vite, React, and Tailwind

## Coding Conventions
1. **KISS** — Prefer the simplest solution. No unnecessary abstractions.
2. **Explicit over implicit** — Extract constants, name lambdas, use JSDoc annotations.
3. **Functional-first** — Prefer pure functions, explicit lambdas, and functional composition.
4. **Minimal tooling** — The dependency list is intentionally small. Avoid adding libraries unless strictly necessary. Prefer long-lived, ubiquitous tools (Lindy Effect).
5. **Organized and legible** — Use "extract constant" refactors liberally. Group related logic. Keep files focused on a single responsibility.

### Code Patterns
- JSDoc comments on all public methods
